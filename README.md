# ShopSphere E-Ticaret Projesi

ShopSphere, modern web teknolojileri kullanılarak geliştirilmiş bir e-ticaret uygulamasıdır.

## 📸 Ekran Görüntüleri

### Ana Sayfa
![Ana Sayfa](./ShopSphereUI/public/images/anasayfa.png)
Modern ve etkileyici bir ana sayfa tasarımı ile kullanıcıları karşılıyoruz. Gradient arka plan ve animasyonlu geçişler ile dinamik bir deneyim sunuyoruz.

### Hakkımızda
![Hakkımızda](./ShopSphereUI/public/images/hakkimizda.png)
Ekibimizi ve vizyonumuzu modern bir tasarımla tanıtıyoruz. Hover efektleri ve yumuşak animasyonlar ile kullanıcı deneyimini zenginleştiriyoruz.

### İletişim
![İletişim](./ShopSphereUI/public/images/iletişim.png)
Kullanıcı dostu iletişim formu ve modern tasarım ile müşterilerimize kolay erişim imkanı sağlıyoruz.

## 🛠️ Kullanılan Teknolojiler

### Backend
- .NET 9.0
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server

### Frontend
- React.js 18
- TypeScript
- Material-UI (MUI v6)
- Framer Motion
- React Router v7

## 🌟 Özellikler

- Modern ve responsive tasarım
- Animasyonlu sayfa geçişleri
- Material Design tabanlı UI
- TypeScript ile tip güvenliği
- SEO dostu yapı

## 🚀 Kurulum

1. Backend için:
```bash
cd ShopSphere.API
dotnet restore
dotnet run
```

2. Frontend için:
```bash
cd ShopSphereUI
npm install
npm run dev
```
