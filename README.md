## 🛠️ Technologies Used

### Backend
- .NET 9.0
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server

### Frontend
- React.js 18
- TypeScript
- Material-UI (MUI v6)
- Framer Motion
- React Router v7

## 🚀 Installation

1. For Backend:
```bash
cd ShopSphere.API
dotnet restore
dotnet run
```

2. For Frontend:
```bash
cd ShopSphereUI
npm install
npm run dev
```
